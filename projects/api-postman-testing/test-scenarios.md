# API-сценарии

## Авторизация

| ID | Endpoint | Метод | Сценарий | Ожидаемый результат | Приоритет |
| --- | --- | --- | --- | --- | --- |
| API-001 | `/auth/login` | POST | Авторизация с валидными учетными данными | `200 OK`, ответ содержит `accessToken`, `refreshToken`, `expiresIn` | Высокий |
| API-002 | `/auth/login` | POST | Авторизация с неверным паролем | `401 Unauthorized`, код ошибки `INVALID_CREDENTIALS` | Высокий |
| API-003 | `/auth/login` | POST | Отправка запроса без пароля | `400 Bad Request`, ошибка валидации указывает на поле `password` | Высокий |
| API-004 | `/users/me` | GET | Запрос без токена | `401 Unauthorized`, персональные данные не возвращаются | Высокий |
| API-005 | `/users/me` | GET | Запрос с истекшим токеном | `401 Unauthorized`, код ошибки `TOKEN_EXPIRED` | Средний |

## Каталог

| ID | Endpoint | Метод | Сценарий | Ожидаемый результат | Приоритет |
| --- | --- | --- | --- | --- | --- |
| API-006 | `/products` | GET | Получение списка товаров с пагинацией по умолчанию | `200 OK`, ответ содержит `items`, `total`, `page`, `limit` | Высокий |
| API-007 | `/products?limit=0` | GET | Невалидная нижняя граница `limit` | `400 Bad Request`, ошибка валидации | Средний |
| API-008 | `/products?limit=101` | GET | Невалидная верхняя граница `limit` | `400 Bad Request`, ошибка по максимальному значению `limit` | Средний |
| API-009 | `/products/{id}` | GET | Получение существующего товара | `200 OK`, товар содержит `id`, `name`, `price`, `stock` | Высокий |
| API-010 | `/products/{id}` | GET | Получение несуществующего товара | `404 Not Found`, код ошибки `PRODUCT_NOT_FOUND` | Высокий |

## Корзина

| ID | Endpoint | Метод | Сценарий | Ожидаемый результат | Приоритет |
| --- | --- | --- | --- | --- | --- |
| API-011 | `/cart/items` | POST | Добавление товара в корзину | `201 Created`, товар появляется в корзине, количество корректное | Высокий |
| API-012 | `/cart/items` | POST | Добавление товара с количеством `0` | `400 Bad Request`, ошибка валидации | Высокий |
| API-013 | `/cart/items` | POST | Добавление товара в количестве больше остатка | `409 Conflict`, код ошибки `NOT_ENOUGH_STOCK` | Высокий |
| API-014 | `/cart/items/{itemId}` | PATCH | Изменение количества товара | `200 OK`, итоговая сумма корзины пересчитана | Высокий |
| API-015 | `/cart/items/{itemId}` | DELETE | Удаление позиции из корзины | `204 No Content`, после `GET /cart` позиция отсутствует | Средний |

## Заказы

| ID | Endpoint | Метод | Сценарий | Ожидаемый результат | Приоритет |
| --- | --- | --- | --- | --- | --- |
| API-016 | `/orders` | POST | Создание заказа из валидной корзины | `201 Created`, статус заказа `created`, сумма заказа равна сумме корзины | Высокий |
| API-017 | `/orders` | POST | Создание заказа из пустой корзины | `409 Conflict`, код ошибки `EMPTY_CART` | Высокий |
| API-018 | `/orders/{id}` | GET | Получение созданного заказа | `200 OK`, ответ содержит позиции, сумму и историю статусов | Высокий |
| API-019 | `/orders/{id}/cancel` | POST | Отмена созданного заказа | `200 OK`, статус меняется на `cancelled` | Средний |
| API-020 | `/orders/{id}/cancel` | POST | Отмена уже оплаченного заказа | `409 Conflict`, ошибка бизнес-правила | Высокий |

## Контрактные проверки

| ID | Объект | Обязательные поля | Проверки типов |
| --- | --- | --- | --- |
| CT-001 | Товар | `id`, `name`, `price`, `currency`, `stock`, `isAvailable` | `price` number, `stock` integer, `currency` enum |
| CT-002 | Корзина | `id`, `items`, `total`, `currency` | `items` array, `total` number |
| CT-003 | Заказ | `id`, `status`, `items`, `total`, `createdAt` | `status` enum, `createdAt` ISO date |
| CT-004 | Ошибка | `code`, `message`, `traceId` | `code` string, `traceId` string |
