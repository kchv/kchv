# Баг-репорты: страница поиска

## BR-001: Отсутствует сортировка в разделе документов

| Поле | Значение |
| --- | --- |
| Серьезность | Major |
| Приоритет | Высокий |
| Окружение | Windows 10, Chrome |
| Статус | Открыт |
| Найдено в | Release candidate страницы поиска |

### Описание

При переходе в раздел документов результаты невозможно отсортировать. Если результатов много, пользователь не может быстро перейти к самым актуальным материалам.

### Шаги воспроизведения

1. Открыть страницу поиска.
2. Ввести релевантный запрос с большим количеством результатов.
3. Перейти в раздел документов.

### Фактический результат

Инструменты сортировки отсутствуют.

### Ожидаемый результат

Отображаются варианты сортировки, например "По релевантности" и "Сначала новые".

### Влияние

Пользователь не может быстро найти актуальные документы, если выдача содержит много результатов.

## BR-002: Количество результатов в разделе документов становится равным нулю после перехода в форум

| Поле | Значение |
| --- | --- |
| Серьезность | Medium |
| Приоритет | Высокий |
| Окружение | Windows 10, Chrome |
| Статус | Открыт |
| Найдено в | Release candidate страницы поиска |

### Описание

После перехода в раздел форума счетчик результатов в разделе документов отображается как `0`, хотя до переключения в разделе были результаты.

### Шаги воспроизведения

1. Открыть страницу поиска.
2. Ввести релевантный запрос с большим количеством результатов.
3. Перейти в раздел форума.
4. Проверить счетчик результатов в разделе документов.

### Фактический результат

Количество найденных результатов в разделе документов отображается равным `0`.

### Ожидаемый результат

Количество результатов остается консистентным при переключении разделов и страниц.

### Влияние

Счетчик вводит пользователя в заблуждение и снижает доверие к результатам поиска.

## BR-003: Изменяется количество результатов в разделе форума при переходе между разделами

| Поле | Значение |
| --- | --- |
| Серьезность | Major |
| Приоритет | Высокий |
| Окружение | Windows 10, Chrome |
| Статус | Открыт |
| Найдено в | Release candidate страницы поиска |

### Описание

При переходе между разделами меняется общее количество результатов в разделе форума. Из-за этого пагинация работает некорректно: при переходе на последнюю страницу система может не находить результаты.

### Шаги воспроизведения

1. Открыть страницу поиска.
2. Ввести релевантный запрос с большим количеством результатов.
3. Перейти в раздел форума.
4. Несколько раз переключиться между разделами.
5. Перейти на последнюю страницу форума.

### Фактический результат

Количество результатов в разделе форума меняется при переходе между разделами.

Пример технического запроса:

```http
GET /async/search?query=<term>&type=forum
```

### Ожидаемый результат

Количество результатов остается одинаковым вне зависимости от переходов, пагинация работает корректно.

### Влияние

Пользователь может попасть на пустую последнюю страницу и решить, что часть результатов недоступна.
