# Тест-кейсы: страница поиска

## Функциональные проверки

| ID | Название | Приоритет | Предусловия | Шаги | Ожидаемый результат | Фактический результат | Статус |
| --- | --- | --- | --- | --- | --- | --- | --- |
| TC-001 | Поиск по релевантному запросу | Высокий | Страница поиска открыта | 1. Ввести запрос с ожидаемыми результатами. 2. Нажать Enter. | Отображается список результатов, строка поиска сохраняет введенный запрос, URL содержит параметры поиска. | Соответствует ожидаемому. | Пройден |
| TC-002 | Переход по разделам после поискового запроса | Высокий | Есть результаты в нескольких разделах | 1. Перейти в раздел "Форум". 2. Перейти в раздел "Нормативные документы". 3. Вернуться в "Новости и статьи". | Открываются нужные разделы, состояние поиска сохраняется. | Соответствует ожидаемому. | Пройден |
| TC-003 | Консистентность счетчиков результатов | Высокий | Есть результаты в нескольких разделах | 1. Выполнить поиск. 2. Зафиксировать счетчики разделов. 3. Переключиться между разделами 3-5 раз. 4. Сравнить счетчики. | Счетчики не меняются без изменения запроса, фильтров или данных. | В части разделов счетчики меняются или обнуляются. | Не пройден |
| TC-004 | Пагинация списка результатов | Высокий | Запрос возвращает больше одной страницы | 1. Проверить первую страницу. 2. Нажать "Следующая". 3. Нажать "Предыдущая". 4. Перейти по номеру страницы. | Выдача соответствует выбранной странице, дубликатов в пределах соседних страниц нет. | Соответствует ожидаемому. | Пройден |
| TC-005 | Прямой переход по URL с номером страницы | Высокий | Известен URL с параметрами поиска | 1. Скопировать URL второй страницы. 2. Открыть в новой вкладке. | Открывается та же страница выдачи, query и активный раздел восстановлены. | Соответствует ожидаемому. | Пройден |
| TC-006 | Отрицательный номер страницы в URL | Средний | Нет | 1. Открыть `/search?query=<term>&type=articles&page=-3`. | Система безопасно обрабатывает невалидный номер: редиректит на первую страницу или показывает корректный fallback. | Соответствует ожидаемому. | Пройден |
| TC-007 | Отсутствие результатов | Средний | Нет | 1. Ввести нерелевантный запрос. 2. Проверить каждый раздел. | Отображается понятное сообщение, нет пустого белого экрана или ошибки сервера. | Соответствует ожидаемому. | Пройден |
| TC-008 | Сортировка "Сначала новые" | Высокий | Запрос возвращает результаты с датами | 1. Выбрать сортировку по дате. 2. Проверить первую, вторую и последнюю страницу. 3. Повторить в разделах. | Результаты отображаются от новых к старым во всех применимых разделах. | В одном разделе сортировка отсутствует. | Не пройден |
| TC-009 | Сортировка "По релевантности" | Высокий | Запрос возвращает разные типы результатов | 1. Выбрать сортировку по релевантности. 2. Перейти на следующую страницу. 3. Вернуться назад. | Сортировка сохраняется, порядок выдачи соответствует выбранному режиму. | В одном разделе сортировка отсутствует. | Не пройден |
| TC-010 | Расширенный поиск в разделе форума | Высокий | Выполнен поиск | 1. Перейти в "Форум". 2. Открыть расширенные фильтры. | Отображаются фильтры по периоду, области поиска, разделу и автору. | Соответствует ожидаемому. | Пройден |
| TC-011 | Автозаполнение автора | Средний | Есть справочник авторов | 1. Начать ввод имени существующего автора. 2. Начать ввод несуществующего автора. | Для существующего автора есть варианты, для несуществующего отображается "Не найдено". | Соответствует ожидаемому. | Пройден |
| TC-012 | Несовместимые фильтры форума | Высокий | Выполнен поиск | 1. Выбрать раздел. 2. Указать несуществующего автора. 3. Нажать "Найти". | Отображается корректное пустое состояние без ошибки сервера. | Соответствует ожидаемому. | Пройден |

## Валидация ввода и граничные случаи

| ID | Название | Приоритет | Тестовые данные | Ожидаемый результат | Статус |
| --- | --- | --- | --- | --- | --- |
| TC-013 | Поиск с пробелами в начале и конце | Средний | `"  НДС  "` | Пробелы обрезаются, поиск выполняется по значимому тексту. | Пройден |
| TC-014 | Поиск только из пробелов | Высокий | `"     "` | Поиск не выполняется или отображается валидное пустое состояние. | Пройден |
| TC-015 | HTML-теги в поисковом запросе | Высокий | `<script>alert(1)</script>` | Теги не исполняются, ошибки сервера нет. | Пройден |
| TC-016 | SQL-like ввод | Высокий | `' OR 1=1 --` | Поиск не возвращает все записи, ошибки сервера нет. | Пройден |
| TC-017 | Длинный запрос | Средний | 500+ символов | Интерфейс не ломается, текст ограничивается или корректно обрабатывается. | Пройден |
| TC-018 | Смешанная кириллица/латиница | Средний | `HДС` | Система возвращает ожидаемый результат или корректное отсутствие результатов. | Пройден |
