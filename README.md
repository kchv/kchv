## QA Engineer

QA Engineer с опытом ручного тестирования web/API/mobile, подготовки тестовой документации, анализа требований, работы с багами, логами и пользовательскими сценариями.

Сильные стороны: системный подход к проверкам, внимательность к деталям, умение разбирать сложные сценарии, оформлять понятные тестовые артефакты и фиксировать дефекты так, чтобы команда могла быстро их воспроизвести и исправить.
